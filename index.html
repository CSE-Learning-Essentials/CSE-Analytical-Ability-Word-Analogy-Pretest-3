SECTION 2: ANALYTICAL ABILITY

PART 1: WORD MEANING

10-ITEM TEST | PASSING SCORE: 8

Directions: Each question below consists of a related pair of words. From among the suggested answers, choose the word or group of words that best expresses a similar relationship from the original pair.

1. DECEMBER : CHRISTMAS ::

2. EVIDENCE : VERDICT ::

3. TELESCOPE : OBSERVE ::

4. STUDY : SUCCESS ::

5. FEATHER : LIGHT ::

6. KEY is to LOCK as PASSWORD is to ________

7. OXYGEN is to LIFE as FUEL is to ________

8. ATLAS is to MAPS as DICTIONARY is to ________

9. HEART is to PUMPING as LUNGS is to ________

10. BYTE is to KILOBYTE as PAGE is to ________

Answer Explanations:

  1. B. January : New Year
    Relationship: Time to Event.
    DECEMBER : CHRISTMAS :: JANUARY : NEW YEAR
    Explanation: Christmas is celebrated in December; New Year is celebrated in January. Both pairs show a month-to-holiday relationship.
  2. E. experiment : result
    Relationship: Reason to Conclusion.
    EVIDENCE : VERDICT :: EXPERIMENT : RESULT
    Explanation: Evidence leads to a verdict in court; an experiment leads to a result in science. Both pairs show cause-effect or basis-to-outcome relationships.
  3. D. microscope : magnify
    Relationship: Tool to Purpose.
    TELESCOPE : OBSERVE :: MICROSCOPE : MAGNIFY
    Explanation: A telescope is used to observe distant objects; a microscope is used to magnify tiny objects. Both pairs show a tool/instrument and its intended purpose.
  4. E. exercise : fitness
    Relationship: Condition to Outcome.
    STUDY : SUCCESS :: EXERCISE : FITNESS
    Explanation: Studying can lead to success; exercising can lead to fitness. Both pairs show an action leading to a desired outcome.
  5. B. silk : soft
    Relationship: Object to Characteristic.
    FEATHER : LIGHT :: SILK : SOFT
    Explanation: A feather is light; silk is soft. Both pairs show an object and its inherent quality or property.
  6. E. access
    Relationship: Tool to Purpose.
    KEY : LOCK :: PASSWORD : ACCESS
    Explanation: A key opens a lock; a password grants access. Both pairs show an item that provides entry or functionality.
  7. D. engine
    Relationship: Substance to Effect/Use.
    OXYGEN : LIFE :: FUEL : ENGINE
    Explanation: Oxygen sustains life; fuel powers an engine. Both pairs show a necessary input that enables function or activity.
  8. D. definitions
    Relationship: Object to Content.
    ATLAS : MAPS :: DICTIONARY : DEFINITIONS
    Explanation: An atlas contains maps; a dictionary contains definitions. Both pairs show a container/object and the content it holds.
  9. C. breathing
    Relationship: Part to Function.
    HEART : PUMPING :: LUNGS : BREATHING
    Explanation: The heart’s function is pumping blood; the lungs’ function is breathing. Both pairs show an organ and its primary function.
  10. A. chapter
    Relationship: Part to Whole.
    BYTE : KILOBYTE :: PAGE : CHAPTER
    Explanation: A byte is part of a kilobyte; a page is part of a chapter. Both pairs show a smaller unit forming part of a larger structure.
